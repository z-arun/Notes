NOR vs NAND:

NOR:
memory cells connected in parallel
low density
fast read
slow write 
can access any byte
used for FW storage.
high cost

NAND;
Memory cells connected in serial
fast write 
slow read
used for more storage
high density 
low cost

-------------------------------------------------->>
Page  - smallest mem unit that can be written/read 
Block - smallest unit that can be erased, it will be group of pages
